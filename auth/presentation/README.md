# Auth: Presentation Module

This module is responsible for the UI of the authentication feature. It contains the screens for the intro, login, and register flows, as well as the view models and other UI-related components.

## Features

- **Intro:** A screen that introduces the user to the app and prompts them to either log in or register.
- **Login:** A screen that allows the user to log in to their account.
- **Register:** A screen that allows the user to create a new account.
- **Auth-specific composables:** A collection of custom composables that are used in the authentication screens.

## Dependencies

- `auth:domain`: This module depends on the `auth:domain` module for authentication-specific domain models and interfaces.
- `core:domain`: This module depends on the `core:domain` module for domain models.
