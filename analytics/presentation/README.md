# Analytics: Presentation Module

This module is responsible for the UI of the analytics feature. It contains the screen for the analytics dashboard, as well as the view model and other UI-related components.

## Features

- **Analytics Dashboard:** A screen that displays analytics about the user's runs, such as total distance, total time, and average speed.
- **Analytics-specific composables:** A collection of custom composables that are used in the analytics screen.

## Dependencies

- `analytics:domain`: This module depends on the `analytics:domain` module for analytics-specific domain models and interfaces.
