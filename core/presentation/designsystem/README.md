# Core: Presentation: Design System Module

This module contains the design system for the Runique application. It defines the colors, typography, and custom composables that are used throughout the app to ensure a consistent look and feel.

## Features

- **Theme:** The main theme for the application, which is based on Material 3.
- **Colors:** A set of custom colors that are used in the theme.
- **Typography:** A set of custom typography styles that are used in the theme.
- **Icons:** A collection of custom icons that are used throughout the app.
- **Components:** A collection of custom composables that can be reused across the application, such as buttons, text fields, and dialogs.

## Dependencies

- **Jetpack Compose:** This module uses Jetpack Compose for building the UI.
- **Material 3:** This module uses the Material 3 library for the design system.
