# App Module

This is the main module of the Runique application. It is responsible for bringing together all the other modules and providing the entry point for the app.

## Features

- **MainActivity:** The main activity of the application, which hosts the Jetpack Compose UI.
- **RuniqueApp:** The main composable that sets up the theme and the navigation for the app.
- **NavigationRoot:** The composable that defines the navigation graph for the app.
- **MainViewModel:** The main view model of the application, which is responsible for handling the overall state of the app.

## Dependencies

This module depends on all the other modules in the project.
